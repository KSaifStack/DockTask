# Features

- Smart Task Organization: Automatically categorize tasks by levels of importance and groups.
- Context-Aware Reminders: Never miss a deadline with OS level notifications.
- Precise Due Dates: Schedule tasks due dates within seconds of each other.
- Custom UI: Built with JavaFX and CSS.  

------------------------------------------------------------------------

# Tech Stack

-   Language: Java
-   Framework: JavaFX, CSS
-   Topics: Application, Calendar, To-Do List, Task Management

# Installation 

**Important:**  
You must download [JDK 23](https://jdk.java.net/23/) and place it inside the `PlanForge` directory before running the application.

1.  Clone the repository:

    ```sh
    git clone https://github.com/KSaifStack/PlanForge.git
    cd PlanForge
    ```

2.  Download JDK 23 from [jdk.java.net/23](https://jdk.java.net/23/) and extract it directly into the `PlanForge` folder.
